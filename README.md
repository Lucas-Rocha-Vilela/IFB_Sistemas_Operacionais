## Ementa

Histórico, conceito e tipos de sistemas operacionais. Conceito de Processo. Gerência de Processos/Processador. Comunicação, Concorência e Sincronização de Processos. Gerenciamento de Memória: Memória Virtual, Paginação, Segmentação e "Swap". Gerenciamento de Arquivos. Gerenciamento de Dispositivos de Entrada/Saída. Alocação de Recursos.
